# Nix Helper Skill (サンプル)

Nix環境の管理をサポートするサンプルスキルです。

## 機能

- Nix設定の再ビルド方法を提供
- パッケージのアップデート方法を提供
- パッケージ検索方法を提供

## 使用方法

Claude Code で以下のように呼び出します:

```
Nixの設定をrebuildしたい
```

```
パッケージをupdateする方法は?
```

## カスタマイズ

`skill.ts` を編集して、独自のNixヘルパー機能を追加できます。

## 注意

これはサンプル実装です。実際の環境に合わせてカスタマイズしてください。
