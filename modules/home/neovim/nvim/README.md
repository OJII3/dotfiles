# My Neovim Settings

## Requirements

- Neovim
- Node.js
- Deno
- SKK Dictionary
- ripgrep
- fzf
- tig
- Marp CLI

## Markdown Presentation

- `nwiizo/marp.nvim` is loaded for Markdown buffers.
- `:MarpWatch` starts live preview for the current Markdown file.
- `:MarpExport [html|pdf|pptx|png|jpeg]` exports the current presentation.
